## 1. Version Bump Rationale

- Milestone label `v1.6` is planning metadata, not npm semver.
- Current published/package version is `@pixelml/agenticflow-cli@1.3.1`.
- When v1.6 closes, ship as **`1.4.0`**.
- Rationale: `1.3.1 -> 1.4.0` is a **minor bump** because the milestone adds new user-facing CLI capability, starting with `af company diff`.

## 2. CHANGELOG Draft

```md
## 1.4.0

- Added `af company diff <file>` to compare a local company export against the live workspace before import.
- Added human-readable diff output for company config changes: `+` new, `~` modified, `<` remote-only, `✓` in sync.
- Added machine-readable `--json` output for company diff with schema `agenticflow.company.diff.v1`.
- Added structured error handling for missing files and invalid YAML in company diff flows.
- Prepared company import merge follow-up work for safer conflict-aware import behavior.
```
