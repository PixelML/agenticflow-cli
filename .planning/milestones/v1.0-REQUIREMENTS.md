# Requirements: AgenticFlow CLI

**Defined:** 2026-04-05
**Core Value:** Any AI can go from `npm install` to useful agent output in under 5 minutes

## v1 Requirements

### Action Workflows

- [ ] **ACT-01**: Workflows chain LLM → `mcp_run_action` to post to Google Business, Instagram, etc.
- [ ] **ACT-02**: CLI detects missing MCP connections and shows `_links.mcp` with setup instructions
- [ ] **ACT-03**: `af connections list --json` shows available connections with categories
- [ ] **ACT-04**: Workflow templates in skills packs use real platform nodes (web_scraping, mcp_run_action, api_call)

### Web UI Integration

- [ ] **WEB-01**: All CLI outputs include `_links` with correct agenticflow.ai URLs
- [ ] **WEB-02**: URLs verified against WorkflowChef-Web routes (agents, threads, connections, mcp, settings)
- [ ] **WEB-03**: Bootstrap `_links` includes connections + MCP management pages

### Ishi Integration

- [ ] **ISHI-01**: Ishi CLI users can use AgenticFlow agents via `af` commands
- [ ] **ISHI-02**: AgenticFlow skill for Ishi (minimal — CLI handles the heavy lifting)

### Business Packs

- [ ] **PACK-01**: amazon-seller-pack v3 with Gemma 4 company blueprint
- [ ] **PACK-02**: At least 2 more business packs (tutor, cafe, freelancer, or clinic)
- [ ] **PACK-03**: Each pack has company.yaml with agents, system prompts, starter tasks

### Quality

- [ ] **QA-01**: Fresh zero-context agent test passes (install → bootstrap → useful output)
- [ ] **QA-02**: All edge cases from hostile QA pass (67 tests)
- [ ] **QA-03**: Autoresearch score >= 8.5/10

## v2 Requirements

### Platform Depth

- **PLAT-01**: Interactive chat mode (`af agent chat`)
- **PLAT-02**: Cost/token tracking per agent
- **PLAT-03**: Workflow execution monitoring (`af workflow watch`)
- **PLAT-04**: Agent cloning (`af agent clone`)

### Ecosystem

- **ECO-01**: Published first-party skills in `af skill list`
- **ECO-02**: Pack marketplace browsing from CLI
- **ECO-03**: Company import/export format

## Out of Scope

| Feature | Reason |
|---------|--------|
| Runtime execution | AgenticFlow platform handles this — CLI is remote control |
| Paperclip source changes | External client, we're compatible not coupled |
| Browser OAuth | CLI uses API keys, web UI handles OAuth |
| Mobile app | CLI is for AI agents with shell access |
