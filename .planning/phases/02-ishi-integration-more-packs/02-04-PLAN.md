Supported pack source formats (from packages/cli/src/cli/pack-registry.ts):
  - github:owner/repo              (GitHub clone, no subpath)
  - github:owner/repo/sub/path     (GitHub clone + copy subpath)
  - https://github.com/owner/repo  (GitHub HTTPS URL)
  - /absolute/local/path           (local copy)
  - ./relative/path                (local copy, resolved from CWD)
